## Library artifacts

All library modules share a single version and are published under the
`com.synapselib` group:

| Artifact          | Current version | Purpose                                              |
|-------------------|-----------------|------------------------------------------------------|
| `core`            | 1.0.12          | Reactive operators for Kotlin Flow                   |
| `arch`            | 1.0.12          | Architecture framework — SwitchBoard, Nodes, Providers, Interceptors |
| `arch-hilt`       | 1.0.12          | Hilt integration for multi-module provider aggregation |
| `arch-test`       | 1.0.12          | Testing utilities — SynapseTestRule, capture helpers  |

```kotlin
dependencies {
    implementation("com.synapselib:arch:1.0.12")

    // Multi-module Hilt projects
    implementation("com.synapselib:arch-hilt:1.0.12")

    // Testing
    testImplementation("com.synapselib:arch-test:1.0.12")
}
```
